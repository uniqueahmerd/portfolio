# Ahmad Usman - Portfolio

Personal portfolio website showcasing my work, skills, and experience as a software developer. Built with modern web technologies and features a bilingual interface (English).
## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/uniqueahmerd/ahmadusman.com.git
cd ahmadusman.com
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
